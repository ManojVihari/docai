# API: get_users

## Overview
The `get_users` function retrieves a list of user objects, with an optional limit parameter to control the number of users returned. By default, it returns up to 10 users.

## Endpoint
- Method: GET
- Path: /users

## Request

### Request Body
| Field | Type | Required | Default |
| --- | --- | --- | --- |
| limit | int | No | 20 |

## Response

### Error Responses
| Code | Meaning |
| --- | --- |
| 404 | User not found |

## Business Logic
This function interacts with the underlying data storage system (e.g., database) to fetch user records based on the provided `limit`. It retrieves the required number of user objects and returns them as a list.

## Change Impact
Modifying the `limit` parameter will affect the number of users returned by the function. A lower value will result in fewer users being fetched, while a higher value will increase the number of users returned.
